# Nusantara Bersuara

Website pengenalan lagu daerah Indonesia yang bertujuan membantu pengguna mengenal dan melestarikan budaya Nusantara melalui lagu-lagu daerah dari berbagai provinsi di Indonesia.

## Fitur

- Menampilkan daftar lagu daerah Indonesia
- Informasi asal daerah
- Informasi pencipta lagu
- Pencarian lagu
- Tampilan responsif

## Teknologi

- HTML5
- JavaScript
- Tailwind CSS

## Kelompok

## Pengembang Website

LaannX

## Cara Menjalankan

1. Download atau clone repository.
2. Buka file `index.html`.
3. Jalankan menggunakan browser.
